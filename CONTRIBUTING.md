# 贡献指南

感谢你考虑为 `nonebot-plugin-dst-qq` 项目做出贡献！

## 🚀 快速开始

### 1. Fork 项目
1. 访问 [项目页面](https://github.com/uitok/nonebot-plugin-dst-qq)
2. 点击右上角的 "Fork" 按钮
3. 克隆你的 Fork 到本地

### 2. 创建分支
```bash
git checkout -b feature/your-feature-name
```

### 3. 提交更改
```bash
git add .
git commit -m "feat: 添加新功能"
```

### 4. 推送更改
```bash
git push origin feature/your-feature-name
```

### 5. 创建 Pull Request
在 GitHub 上创建 Pull Request，描述你的更改。

## 📋 贡献类型

- 🐛 **Bug 修复**
- ✨ **新功能**
- 📚 **文档改进**
- 🔧 **代码优化**
- 🧪 **测试用例**

## 🎯 开发指南

### 代码风格
- 遵循 PEP 8 规范
- 使用有意义的变量和函数名
- 添加适当的注释和文档字符串

### 提交信息
使用清晰的提交信息格式：
- `feat:` 新功能
- `fix:` 修复问题
- `docs:` 文档更新
- `style:` 代码格式
- `refactor:` 代码重构
- `test:` 测试相关
